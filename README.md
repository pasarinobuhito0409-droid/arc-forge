# ARC FORGE

溶接 ── 原理から設計指示まで・朝15分道場（スマホPWA）

- 遊ぶ: https://pasarinobuhito0409-droid.github.io/arc-forge/
- 型: 予想 → 3D実験（Three.js）→ 種明かし（説明カード1枚＝専用3Dアニメ）→ ミニクイズ → スタンプ
- 第1章「溶接を見る→指示する」全6回＋ボス戦収録（アークとは・溶融池とビード・歪みとHAZ・溶接記号・脚長とのど厚・欠陥と検査）
- ⚔ 設計判断ドリル：現場カード24問をOK/NG即断。間違えた問題は復習キューへ
- ⚡ 復習キュー：間隔反復（簡略SM-2）で忘れかけた頃に再登場。1日上限5問
- 🔬 金属の目：原子格子が溶けてバラけ→混ざり→冷えて再結合する様子を見て「溶接＝原子レベルで1つになる」を体感する
- 🗺 溶接マスターマップ：Stage1（基礎）〜Stage4（溶接構造の最適化）までの長期ロードマップ
- 育つ溶接ロボセル：学ぶほどホーム画面のロボが0部品→8部品に育ち、全クリで自動溶接を始める
- 第2章「板金の溶接設計」は次のアップデートで配信予定
