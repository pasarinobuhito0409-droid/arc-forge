# ARC FORGE — 溶接の3D教科書

スマホで触って溶接の原理を学ぶ、インタラクティブ3D教材（PWA）。

## 実験室
- **01 アークの原理** — プラズマ・電離・電子の流れ（電流スライダー）
- **02 溶融池とビード** — TIGトーチ／うろこビード／裏波
- **03 結晶と熱・HAZ** — 鉄の原子格子が振動し溶ける（温度スライダー／BCC・FCC切替）
- **04 継手の種類** — 突合せ／すみ肉／重ね
- **クイズ道場** — 溶接基礎7問（採点＋解説）

## 技術
Three.js (WebGL) によるリアルタイム3D。PBR金属マテリアル・環境マップ反射・影・UnrealBloom。
オフライン対応の Service Worker / PWA manifest 付き。

## 使い方
スマホ／PCのブラウザで開く。ドラッグで回転、スクロールで拡大。
ホーム画面に追加するとアプリとして起動。
